-Configure Git Identity

$ git config --global user.name "izancode"

$ git config --global user.email "izancode11@gmail.com"

-Configure Code Editor

$ git config --global core.editor "code --wait"

-Configure Default Branch

$ git config --global init.defaultBranch main

-List all Configurations

$ git config --list

-Initialization Commands

$ git init

-Add text and with that create a file

$ echo "# jane-story-series" >> README.md

-Add git to staging area

$ git add . (if you want to update the whole folder or file)

$ git add {Folder or File name} (To add a specific folder or file)

-Add commit to Respository(Github) area

$ git commit -m "first commit"

-Check on which branch are you

$ git branch

-Change Branch name eg: master to main

$ git branch -M main

-To connect our local system Folder or File to Github Website

$ git remote add origin https://github.com/izancode/upgrad-course.git (This is http url)

-This is for push the code on the github account

$ git push origin main

$ git push -u origin main (if you want to commit onlyt main branch)

then use

$ git push

-Check the status on which stage is file are there

$ git status

-Comparing Changes using Git Diff

$ git diff --staged (Compare changes with last commit)

-Checking Commit History

$ git log

$ git log -p (This shows difference intoduced in each commit)

$ git log -1 (limit the git log commit to show)

$ git log --oneline (to show all the commit message only)

-Removing and moving Content

$ git rm (removes the file also from the working directory)

$ git mv (A thin convenience command to move a file)

$ git mv oldFile newFile (to rename a file in git)

-Git Branching in a Nutshell

$ git branch (To show all the branches are made)

$ git branch jane_tv_series (create a new branch)

$ cat .git/HEAD (to find a branch file path)

$ ls .git/refs/heads (to enter in the folder)

$ cat .git/refs/heads/branchname (to find a hash key of a commit)

$ git checkout jane_tv_series (to create new branch)

$ git checkout -b jane_web_series (to create and enter into new branch)

$ git diff main..jane_tv_series (to compare two branch)

$ git merge jane_tv_series

-Cloning Repositories

$ git clone https://github.com/sumyat-aung/to-do-list.git

$ git commit filename.txt -m "Learning to work with clones"
